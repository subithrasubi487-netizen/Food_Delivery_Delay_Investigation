# Food Delivery Delay Pattern Investigation

## Project Overview

This project analyzes food delivery data to identify factors contributing to delivery delays. Statistical analysis, data visualization, outlier detection, and hypothesis testing were performed to generate business insights and recommendations.

## Objective

To investigate delivery delay patterns and identify the key factors affecting delivery performance.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SciPy

## Project Phases

### Phase 1: Data Understanding

* Loaded dataset
* Checked data types
* Identified missing values
* Identified duplicate records

### Phase 2: Statistical Analysis

* Mean
* Median
* Mode
* Standard Deviation
* Variance
* Quartiles
* Skewness Analysis

### Phase 3: Outlier Investigation

* Box Plot
* IQR Method
* Z-Score Method

### Phase 4: Relationship Analysis

* Correlation Matrix
* Heatmap
* Scatter Plots
* Relationship between Distance, Traffic, Weather, Preparation Time and Delivery Time

### Phase 5: Hypothesis Testing

* ANOVA Test
* Weather Impact Analysis

### Phase 6: Business Insights

* Factors causing delays identified
* Operational recommendations provided

## Key Findings

* Average delivery time: 56.73 minutes
* Delivery time distribution is positively skewed
* Distance, traffic, preparation time, and weather influence delivery performance
* Outliers indicate unusually delayed deliveries

## Recommendations

1. Optimize delivery routes.
2. Reduce restaurant preparation time.
3. Improve delivery partner allocation during peak traffic.
4. Implement weather-based delivery planning.
5. Monitor high-delay orders proactively.

## Conclusion

The analysis identified major factors affecting food delivery delays and provided actionable recommendations to improve operational efficiency and customer satisfaction.
